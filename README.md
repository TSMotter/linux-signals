# linux signals
- Personal studies and examples of linux signals

- To build:
```bash
./bbuild.sh <target>
```

- To build and run:
```bash
./bbuild.sh <target> run
```

- Where "`<target>`" is the name of one of the folders containing source code, like default_signal_handling

- To create a new folder (target):
```bash
./create.sh <name>
```
